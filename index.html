What is Digiex .NET Minecraft Launcher?

Digiex .NET Minecraft Launcher is a custom launcher for Minecraft. It allows you to switch between different versions of Minecraft.

How do I get it?

You can download it for Windows here: http://minecraft.digiex.org/MCLauncher.NET/

I found a bug, where can I report it?

If you find a bug, please report it at our GitHub Issues page. Please browse through existing issues if someone else has reported the bug before.
